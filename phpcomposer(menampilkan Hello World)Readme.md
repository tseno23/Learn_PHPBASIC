# PHP Composer Hello World

Proyek ini belajar menggunakan PHP dengan Composer.

## File
- `helloword.php` → menampilkan Hello World denagn php composer
- `dependdencyManagement.php` → contoh penggunaan dependency

## Cara Jalankan
1. Clone repo
2. Jalankan `composer install`
3. Jalankan file PHP

## Catatan
- Folder `vendor/` dan `src/` di-ignore di Git
- Gunakan PHP versi 8+
